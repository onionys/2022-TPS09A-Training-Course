# 2022 NSRRC TPS09A L4 課程補充教材

下圖為搖晃放置在繞射儀上的晶體，在倒空間中的樣子。
量測進行時透過轉動 th, chi, phi 三個角度(馬達)來轉動晶體。

X-axis, Y-axis, Z-axis 就是講議的 Lab Frame (實驗室座標軸)。
H-axi, K-axis, L-axis 就是講議的 Crystal Frame (晶體座標軸)，其實就是 Miller Index。
